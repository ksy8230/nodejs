## npm 관리 명령어

-   npm ll : 현재 프로젝트에 설치된 npm 라이브러리
-   npm ll -g : 글로벌에 설치된 npm 라이브러리
-   npm ll -g --depth=0 : 글로벌레 설치된 npm 라이브러리들 중 뎁스 0만 출력
-   npm un underscore(라이브러리명): 해당 라이브러리 삭제
